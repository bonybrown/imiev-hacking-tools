# Using an ELM327 to send KWP2000/UDS service (ISO 14230) diagnostic protocol requests

### The setup

* Reset the device
* Set CAN protocol
* Set the PID of sent data to 751 (addresses the EV-ECU)
* Set the Flow Control address to 751
* Set the recieve address to 752
* Set the Flow Contol data to `30 00 00` (send as much data as you have with no delay)
* Set the Flow control mode to 1 (use the flow control data above)

```
atz
atsp6
atsh 751
atfcsh 751
atcra 752
atfcsd 30 00 00
atfcsm1
```

### Sending a request

eg sending `sid=33 (0x21) ReadDataByLocalIdentifier, lid=36 (0x24)`
```
2124
```

### The response
```
2124
00C 
0: 61 24 64 A2 A2 01 
1: 41 00 DE 00 00 00 55 
```

### The response breakdown

* Byte 0 0x61 is the request service id, echoed back with the "success" bit (0x40) set.
* Byte 1 is the local id, echoed back
* Byte 2 is the first byte of the data returned for the request
```
├─ Item 143 [iid=77000297]: State of charge(control value) [byte_pos=2, byte_len=1, unit=%, scale(factor=0.5, offset=0), enum(off=0, on=1)]
```
*    0x64 = 100 * 0.5 = 50 %. Correct

```
├─ Item 144 [iid=77000298]: Battery Total voltage [byte_pos=3, byte_len=1, unit=V, scale(factor=2, offset=0)]
```
*    0xA2 = 162 * 2 = 324 V. Correct

```
├─ Item 145 [iid=77000299]: Condenser voltage [byte_pos=4, byte_len=1, unit=V, scale(factor=2, offset=0)]
```

*    0xA2 = 162 * 2 = 324 V. Correct

```
├─ Item 146 [iid=77000300]: Shift position(control value) [byte_pos=5, byte_len=1, scale(factor=1, offset=0)]
```
*    0x01 = 1. Vehicle is in park. Shifting to neutral changes this byte to 0x03

```
├─ Item 147 [iid=77000301]: Coolant water temperature [byte_pos=6, byte_len=1, unit=°C, scale(factor=1, offset=-40)]
```
*    0x41 = 65 - 40 = 25°C. Seems right

```
├─ Item 148 [iid=77000302]: WP speed [byte_pos=7, byte_len=1, unit=r/min, scale(factor=50, offset=0)]
```
* 0x00 = 0. Water pump is not running. 
* When running, reports 0x5E = 94 * 50 = 4700 r/min. Seems plausible.

```
├─ Item 149 [iid=77000303]: OBC connecting signal voltage [byte_pos=8, byte_len=1, unit=V, scale(factor=0.02, offset=0), enum(shutdown release=0, shutdown=1)]
```
*    0xde = 222 * 0.02 = 4.44V. OBC is not plugged in. Could be right.

```
└─ Item 150 [iid=77000304]: OBC connecting signal [byte_pos=9, byte_len=1]
```
*    0x00 = 0. OBC is not connected.
